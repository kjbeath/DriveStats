For analysing BackBlaze drive stats
Files from BackBlaze https://www.backblaze.com/cloud-storage/resources/hard-drive-test-data should be expanded and placed in the data directory. After building the summary file these may be deleted.

sumbackblaze.csv - The latest summary file created by buildbackblaze

buildbackblaze.R - Reads data from BackBlaze to create a file containing all drives

analysebackblaze.R - Analyses the summary data
